:pypi:`pyandexmap` работа с Яндекс Карты API используя Python
=============================================================

У Яндекс Карт хороший геокодинг и есть API, но нет автодополнение поисковой
строки (как в гугле). У гугла плохой геокодинг (для России) но есть
автодополнение. Более подробно можно прочитать например `здесь
<http://clubs.ya.ru/mapsapi/replies.xml?item_no=17733>`_. Парадокс какой-то. Я
написал простой модуль для вычленения данных через API при помощи
питона. Это очень удобно использовать в AJAX запросах. Пример использования
здесь (:ref:`osm_geotagging`). Код для изучения/исправления здесь
https://github.com/uralbash/pyandexmap.
