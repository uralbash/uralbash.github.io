Обновление sacrud и pyramid_ext
===============================

Новая версия :pypi:`sacrud` ``0.0.3``. В ней поправлены некоторые баги,
добавлены :strike:`нескучные обои` элементы дизайна в расширении для
``Pyramid`` и создан отдельный репозитарий с примерами работы разных типов
полей (pyramid_sacrud_example). Pyramid_sacrud_example работает только с
:l:`Postgres` потому что включает в себя примеры полей специфичных именно для этой
БД (таких как ``hstore``).
