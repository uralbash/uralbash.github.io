Пропал доступ в Интернет в Docker контейнерах
=============================================

Docker начиная с версии ``1.10`` и выше, перестал брать значение ``MTU`` от
сетевого интерфейса. Теперь его нужно указывать явно:

.. code-block:: bash

    $ docker daemon --mtu=1372

Это может быть причиной отсутствия Интернет.
