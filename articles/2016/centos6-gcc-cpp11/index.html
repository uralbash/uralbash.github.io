gcc для C++11 из реп в CentOS6¶

C++11 в gcc появился, начиная с версии 4.7, но в CentOS6 естественно более старая версия без С++11. Как вариант, можно собрать из исходников, но я опишу способ установки из дополнительных реп.

См.также

https://gcc.gnu.org/projects/cxx-status.html#cxx11

Добавляем репу:

$ sudo yum install centos-release-scl

Устанавливаем devtoolset-3-toolchain, который включает в себя gcc, gdb, binutils, elfutils, dwz, memstomp, strace, ltrace:

$ sudo yum install devtoolset-3-toolchain

Прописываем gcc в окружение пользователя:

$ scl enable devtoolset-3 bash

или

$ echo "source /opt/rh/devtoolset-3/enable" >> ~/.bashrc

Профит!
